# Amaya Davis Portfolio

This is my personal portfolio website built with HTML, CSS, and JavaScript.

## About
This project showcases my skills, experience, and projects as a developer. It includes a dynamic skills section, GitHub API integration, and a message form that allows users to leave messages on the page.

## Features
- About section with personal information
- Experience section highlighting past learning and projects
- Skills list generated with JavaScript
- GitHub API project integration
- Contact links (GitHub, LinkedIn, Email)
- Leave a message for with dynamic message display and delete functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM manipulation + Fetch API)

## Projects Included
- Weather App (GitHub linked in portfolio)
